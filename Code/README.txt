What to run? 
main.m file which is provided in the zip file should be ran

Parameters? 
In main.m file, the parameters are shown:
1). Whether the ContraceptionKillsFetus = false or true. For both values of these it will work
Whether the RelocatingPopulation = false or true. For both values of these it will work.

In ElephantControl.m file, the constructor ElephantControl() has the Logistic Distributions' initialization,
Change that to desired value and desired scale to manipulate it for different populations. 
The scale determines how soon should you start darting the population, so both ways have their repurcussions. 

Please be careful in changing those values.
